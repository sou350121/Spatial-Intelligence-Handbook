# Spatial Daily Reports

> Auto-generated by Pulsar Spatial pipeline (`scripts/pulsar/`).
>
> 每個 weekday 一份 markdown，含當天 arxiv cs.RO / cs.CV / cs.AI / cs.LG 過濾後 LLM 評級結果。

## 文件命名

```
YYYY-MM-DD.md
```

## 結構

```markdown
# Spatial Daily — YYYY-MM-DD

> Pulsar pipeline auto-generated. N papers rated; ⚡ X · 🔧 Y · 📖 Z

## ⚡ Load-bearing (X)
- [Title](link) (arxiv_id · category) ⭐
  - reason · [tags]

## 🔧 Engineering (Y)
...

## 📖 Reference (Z)
...
```

## Retention

90 天自動 prune（`scripts/pulsar/post.py` 內建）。需要長期歸檔可手動 grep 出後 commit 到別處。

## Weekly summary

`reports/spatial-weekly/` Phase 2 才開（待 ~30 daily 累積後再寫聚合腳本）。

## 整合方式

- TG: 即時推送 top 5 ⚡/🔧 picks
- Handbook git: 每個 daily markdown 文件 commit 進 git（CI 可選擇是否 auto-commit）
- Mintlify: 自動 nav 加入（gen_mintlify_nav.py 已 covers reports/）

詳見 [`scripts/pulsar/README.md`](../../scripts/pulsar/README.md) 與 [`docs/pulsar-integration.md`](../../docs/pulsar-integration.md)。
