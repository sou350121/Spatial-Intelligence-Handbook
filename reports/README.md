# Reports · Pulsar-pipeline 自動產出

| Subdirectory | Cadence | Format |
|---|---|---|
| `weekly/` | Sunday | Forward-looking — surprises, falsifiable hypotheses, watch list |
| `biweekly/` | Every 2 weeks | Retrospective — trends, insights, prediction scorecard |

These mirror VLA-Handbook's `reports/` layout. Re-uses the same pipeline with a `domain=spatial` switch on the hypothesis registry.
